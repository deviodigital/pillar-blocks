== Description ==

The Content Blocks plugin from [Pillar Press](http://pillar.press) provides you with custom blocks for the new [Gutenberg](#?utm_source=WP.org%20Description%20Gberg%20Link&utm_campaign=wporg_gberg_link) block editor.

**Please Note**: Content Blocks is under active development and is used on production sites at your own risk. Because the new editor is still in production, breaking changes could be expected.

== Installation ==

This plugin can be installed directly from your site.

1. Log in and navigate to _Plugins &rarr; Add New_.
2. Type "Pillar Press" into the Search box and hit **Enter**.
3. Locate the **Pillar Press Content Blocks** plugin in the list of search results and click **Install Now**.
4. Once installed, click the Activate link.

It can also be installed manually.

1. Download the **Content Blocks** plugin from WordPress.org.
2. Unzip the package and move to your plugins directory.
3. Log into WordPress and navigate to the Plugins screen.
4. Locate **Content Blocks** in the list and click the *Activate* link.

== Frequently Asked Questions ==

= Can Content Blocks be used with any theme? =

Yes, you can use **Content Blocks** with any theme.

= Do I need Gutenberg to use Content Blocks? =

Yes, until Gutenberg is merged into core, you will need to install the [Gutenberg plugin](https://wordpress.org/plugins/gutenberg/) to use Content Blocks.
